## **//BOT**
> Commands that effect the bot. 
* `txt.invite`, 
* `txt.ping`
## **//FUN**
> Fun commands to waste some time with. 
* `txt.dice`, 
* `txt.rate`
## **//GAMES**
> Commands to play games in the bot.
* `txt.hangman`, 
* `txt.minesweeper`
## **//VC**
> Commands that make bot.txt join your vc and do something.
* `txt.bruh`
## **//OTHER**
> Commands that dont really fit in other boxes. 
* `txt.serverinfo`, 
* `txt.randomcolour`
